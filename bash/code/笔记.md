还可以让脚本本身就具有可执行权限，通过chmod命令可以修改

# 赋予脚本的所有者该执行权限，允许该用户执行该脚本
$ chmod u+rx hello.sh
# 执行命令，这将使用脚本第一行指定的shell来执行，如果指定shell不存在，将使用系统默认shell来执行
$  ./hello.sh

#!是用于指定当前脚本的解释器
